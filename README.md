# LatentGaussianVAR
Code repository for reproducing all simulation results in the paper "Analytic Standard Errors for Latent Gaussian Discrete-Valued Multivariate Time Series" (Preprint)

# Overview
The following files are included in this repository:
- `run_sim.R`: Simulates data and fits all models. 
- `analyze.R`: Calculates summary measures using simulated data and results from `run_sim.R`. 
- `calculate_measures.R`: Function used by `analyze.R` for computing metrics
- `mats.R`: Generates true transition matrices
- `/files`: Stores simulation results calculated by `analyze.R` as `.RDS`
- `/plots`: Stores plots generated by `plots.R`
- `/data`: Link to repository containing all simulated data and results

# Data and Results Access
Simulating data and fitting all models is computationally intensive, so the original study was conducted using a high performance computing cluster. These data and results can be obtained at the following public repository: [Data/Results](https://doi.org/10.5281/zenodo.21072581).
